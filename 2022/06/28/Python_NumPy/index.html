NumPy 라이브러리 설치방법

NumPy란?

  • NumPy: 수치계산, 행렬 연산. 예측.

  • .array : 리스트를 행렬로 바꿔줌.

  • 이미지도 수치로 표현할 수 있다.

  • .reshape : 행렬의 배열을 바꿔줌 (2:3 -> 3:2)

  • 내장 모듈(=라이브러리= 패키지) (X)

  • 별도 라이브러리 설치 필요

  • 구글코랩에서는 별도 라이브러리 설치 불필요
